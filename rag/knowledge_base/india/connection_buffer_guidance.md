# Connection and Arrival Buffer Guidance

## Work backwards
Start with the required arrival time and include every journey component: pickup uncertainty, traffic variability, walking, station or terminal entry, screening, check-in, platform or gate access, transfers, and a final arrival margin.

## Increase when risk is higher
Use more conservative buffers for unfamiliar stations, separate tickets, terminal changes, international travel, accessibility needs, children, older travellers, heavy baggage, peak traffic, or a high-consequence appointment.

## Explain uncertainty
A buffer reduces risk but cannot guarantee arrival. Live weather, traffic, platform changes, and operator disruption should come from tools. Show the user the tradeoff between a larger buffer and additional waiting time.
